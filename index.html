📖 วิธีติดตั้ง Apps Script

  1. เปิด Google Sheet ใหม่
  2. Extensions → Apps Script
  3. วางไฟล์ Code.gs
  4. Run initSheet()
  5. Deploy → New deployment → Web app
  6. Execute as: Me
  7. Who has access: Anyone
  8. Copy URL มาใส่หน้า login
